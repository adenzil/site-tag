  • Tags

    This feature allows you to group websites and tag them.

  • Other features include :
    • Open all list in new tabs.
    • Tag list in multiple groups.
    • Manage individual website and the entire tag.

  • Tab timer

    If you are one of those you spends too much time on a particular website and want to avoid it this feature lets you setup a timer for a website and it will close automatically when the time is up.

  • Other features include :
    • Warning when you are done halfway with the allocated time.
    • Temporary timer for every page. (right click and choose between 1 or 5 minutes and the page will close when the time is up).
